# Circuit Light (Home Assistant Integration)

Custom Home Assistant integration for Circuit Light.

![Circuit Light screenshot](docs/screenshot.png)

## Install (HACS)

1. In Home Assistant, open **HACS**.
2. Add this repository as a **Custom repository** (type: **Integration**).
3. Install **Circuit Light**.
4. Restart Home Assistant.
5. Add the integration via **Settings → Devices & services → Add integration**.

## Manual install

Copy `custom_components/circuit_light/` into your Home Assistant `config/custom_components/` directory and restart Home Assistant.
